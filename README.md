# 📂 Sidebar Navigation Component

A clean, responsive sidebar navigation menu built with **HTML5** and **CSS3**. This project focuses on layout structure, hover effects, and modern web styling.

## 🚀 Features
- **Responsive Design:** Adapts to different screen heights.
- **Hover Transitions:** Smooth animations when interacting with menu items.
- **Icon Integration:** Professional look using simple CSS-based icons/placeholders.
- **Clean Code:** Semantic HTML for better accessibility.

## 🛠 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Flexbox/Grid layout and custom styling.
