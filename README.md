# 📂 Sidebar Navigation Component

A clean, responsive sidebar navigation menu built with **HTML5** and **CSS3**. This project focuses on UI/UX fundamentals, including layout structure, hover effects, and modern web styling.

## 📸 Preview
![Sidebar Preview](./images/sidebar.png)

## 🚀 Features
- **Responsive Design:** Adapts to different screen heights and layouts.
- **Hover Transitions:** Smooth animations when interacting with menu items.
- **Semantic HTML:** Clean, accessible code structure for better SEO and screen readers.
- **Modular CSS:** Organized styles for easy maintenance.

## 🛠 Tech Stack
- **HTML5:** For the core structure.
- **CSS3:** Flexbox layout, custom transitions, and styling.

## ⚙️ How to Run
1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge).

During this project, I strengthened my Level 1 skills in:
1. **CSS Transitions:** Creating a "premium" feel with smooth movement.
2. **The Box Model:** Managing padding and margins to keep navigation items perfectly aligned.
3. **Asset Management:** Learning how to link local images properly within a project structure.
